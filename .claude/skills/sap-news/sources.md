# Источники для мониторинга SAP-news

Классификация по результатам разведки от 2026-04-23. Обновлять этот файл по мере появления новых источников или изменения качества существующих.

## A. Ядро (must-have) — всегда включать в выдачу

| Домен | Чем ценен | Формат URL |
|---|---|---|
| `cnews.ru` | Основной поток ИТ-новостей и кейсов РФ. Много материалов по теме каждый месяц. | `cnews.ru/news/top/...` |
| `tadviser.ru` | База кейсов по компаниям в разделе «Проект:...». Фактология, детали внедрений. | `tadviser.ru/index.php/Проект:...` |
| `comnews.ru` | Индустрия ИКТ, отраслевая аналитика, большие разборы. | `comnews.ru/content/...` |
| `vedomosti.ru` | Деловая аналитика + регуляторика + эксклюзивы на уровне правительства. | `vedomosti.ru/technology/...` |
| `rbc.ru` (включая `trends.rbc.ru`, `companies.rbc.ru`) | Бизнес-ракурс, тренды. | `rbc.ru/technology_and_media/...` |
| `kommersant.ru` | Деловые сделки, крупные истории. Ценно, но реже по теме. | `kommersant.ru/doc/...` |
| `forbes.ru` | Аналитика, критика, авторские колонки про провалы. | `forbes.ru/tekhnologii/...` |
| `habr.com` | Технические разборы. В приоритете — хабы компаний (корпоративные блоги КОРУС, Нубес, Ассино и т.п. с конкретными кейсами). | `habr.com/ru/companies/...`, `habr.com/ru/articles/...` |

## E. Блоги интеграторов и вендоров — ВЫБОРОЧНО

**Правило:** материал с этих доменов включать в дайджест ТОЛЬКО если это кейс с конкретной компанией-клиентом (упоминается в заголовке), а не маркетинг/продающая страница «аналоги SAP».

Известные домены этого типа:
- Интеграторы: `korusconsulting.ru`, `ibs.ru`, `rdv-it.ru`, `intervolga.ru`, `assino.ru`, `wiseadvice-it.ru`, `1c-rarus.ru`, `beringpro.ru`, `itrp.ru`, `it-lite.ru`
- Вендоры российских ERP: сайты Галактики, Корпорации «Парус», «Консист Бизнес Групп» (Турбо ERP), «Бизнес Технологии» (Global ERP), ELMA

## F. ШУМ — НЕ ВКЛЮЧАТЬ

Продающие страницы «топ аналогов / альтернатив / замен SAP» — они не дают новостей, только забивают выдачу.

Отсекать автоматически:
- `kaiten.ru`, `okocrm.com`, `picktech.ru`, `crmindex.ru`, `msplaut.ru`, `a2is.ru`, `clouderp.ru`, `programstore.ru`, `n-dbc.ru`, `binom.systems`, `all-erp.ru`, `inteh.ooo`, `mypl.pro`, `balns.ru`, `erp-it.ru`, `softunion.ru`, `elma365.com/ru/articles/...` (маркетинг), `agora.ru/blog/...` (маркетинг)
