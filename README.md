# SAP News — мониторинг перехода компаний РФ с SAP

Регулярный сбор новостей о том, как российские компании заменяют SAP (и другие западные ERP/CRM/BI) отечественными решениями — кейсы, регуляторика, мнения рынка.

## Устройство

- `.claude/skills/sap-news/` — скил для Claude Code, умеет собирать новости за указанный период, по компании или отрасли, с дедупликацией и классификацией.
- `news-archive/` — архив сохранённых дайджестов (`digest-YYYY-MM-DD.md`) и список уже виденных URL для дедупликации (`seen-urls.txt`).

## Как запустить вручную

Откройте репо в Claude Code и вызовите:

```
/sap-news                              # дайджест за последние 7 дней
/sap-news --from 2026-01-01            # с указанной даты по сегодня
/sap-news --from 2026-01-01 --to 2026-03-31  # за произвольный период
/sap-news --company Росатом            # всё по конкретной компании
/sap-news --sector нефтегаз            # по отрасли
/sap-news --help                       # справка
```

## Автоматический запуск

Скил рассчитан на запуск через `/schedule` (cloud routine от Claude Code) — раз в неделю агент клонирует репо, собирает свежий дайджест, коммитит и пушит обратно.

## Источники

Фиксированный список в `.claude/skills/sap-news/sources.md` — шесть категорий от «must-have» (CNews, TAdviser, ComNews, Ведомости, РБК) до явного шума (продающие страницы «топ аналогов SAP»), которые отсекаются автоматически.
