# news-archive

Архив дайджестов и уже виденных URL для скила `/sap-news`.

## Файлы

- `seen-urls.txt` — список URL, уже попавших в предыдущие дайджесты. Скил проверяет этот файл перед формированием нового дайджеста, чтобы не повторять новости. По одному URL на строку. Не сортировать — так видна хронология добавления.
- `digest-YYYY-MM-DD.md` — сохранённые дайджесты. Имя файла — дата запуска скила.

## Ручные правки

Если какую-то новость нужно «забыть» (чтобы она снова попала в дайджест) — удалите её URL из `seen-urls.txt`.

Если нужно удалить дайджест целиком — просто удалите файл. На логику скила это не влияет.
